# Trustfeed Crowdsale
This is a custom crowdsale contract with [OpenZeppelin](https://openzeppelin.org/).


## Install
Install dependencies 
```
$ npm install
```

## Compile
```
$ truffle compile
```

## Migrate
```
$ truffle migrate
```

## Test
```
$ truffle test
``` 
## Run External Script
```
$ truffle exec <path/to/file.js>
```
## Docs
